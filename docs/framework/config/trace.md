##  默认配置

```toml tab
[trace]
  enable = false
  provider = "jaeger"
  endpoint = "http://localhost:14268/api/traces"
```

```env tab
TRACE_ENABLE=false
TRACE_PROVIDER="jaeger"
TRACE_PROVIDER_ENDPOINT="http://localhost:14268/api/traces"
```
