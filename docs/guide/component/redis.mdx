# Redis 使用

这里我们选择集成: [go-redis](https://github.com/redis/go-redis)

## 配置组件

<Tabs>
  <Tab label="toml">
```toml
[redis]
  endpoints = ["127.0.0.1:6379"]
  db = 0
  username = ""
  password = ""
  trace = true
  enable_metrics = false
```
  </Tab>
  <Tab label="env">
```sh
REDIS_ENDPOINTS="127.0.0.1:6379"
REDIS_DB=0
REDIS_USERNAME=""
REDIS_PASSWORD=""
REDIS_TRACE=true
REDIS_METRIC=false
```
</Tab>
</Tabs>  

## 基本使用

```go
package main

import (
	"fmt"

	"github.com/infraboard/mcube/v2/ioc/config/redis"
)

func main() {
	client := redis.Client()
	fmt.Println(client)
}
```
